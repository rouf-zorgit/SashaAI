## Step-by-Step for Each File

### For UPDATING files (index.ts, utils.ts):
1. In Supabase, click the filename
2. Click **Edit** button
3. Select all (Ctrl+A) and delete
4. Open local file in VS Code
5. Select all (Ctrl+A) and copy (Ctrl+C)
6. Paste in Supabase (Ctrl+V)
7. Click **Save**

### For ADDING NEW files (all others):
1. In Supabase, click **+ Add File**
2. Type exact filename (e.g., `ltm.ts`)
3. Click **Create**
4. Open local file in VS Code
5. Select all (Ctrl+A) and copy (Ctrl+C)
6. Paste in Supabase (Ctrl+V)
7. Click **Save**
